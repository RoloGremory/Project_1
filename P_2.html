    Create Windows USB Installation Media




  1. Go to "Create installation media for Windows," Choose either Windows 10 or Windows 11 based on your needs,

    and enter the official download page. Click "Download now."







  2. Go to the folder where the downloaded file is located on your computer, and run the MediaCreationTool.

    (If the system prompts a User Account Control window, select "Yes.")





  3. Read the application notices and license terms carefully, then click "Accept."




  4. Choose the system language and version, then click "Next."




  5. Select "USB flash drive" and click "Next."




  6. On the ‘Select a USB flash drive’ page, choose the USB flash drive you want to use, then click "Next."

    The system will start downloading and creating the Windows installation media.





  7. When the tool prompts "Your USB flash drive is ready," click "Finish."








    Install Windows 11



  1. Connect the prepared USB installation media to your computer.

  2. Restart the computer and choose to boot from the USB flash drive.

  3. Select your language, time and currency format, keyboard, or input method, then click "Next."




  4. Click "Install now."



  5. If your computer came with an operating system, choose "I don't have a product key." The computer will automatically activate

    Windows when the installation is complete and connected to the Internet.




  6. Choose the operating system version and click "Next."



  7. After reading the Microsoft Software License Terms, check "I accept Microsoft Software License Terms" and click "Next."



  8. Choose "Custom: Install Windows only (advanced)."



  9. This step involves selecting the disk on which Windows will be installed. The screen will list all the disks and their partitions

    on your computer.

    10. If your computer displays multiple disks, delete all partitions on the disk where you want to install Windows.

    Click on each partition and click "Delete" in sequence.



    After deleting all disk partitions, select the "unallocated space" and click "Next."




  10. After installation completed, the computer will restart and undergo basic Windows setup. Follow the on-screen prompts to complete the setup.



  11. Once setup is complete, the desktop will be shown, and the system installation is finished.

    12. Differences between Windows 10 & Windows 11



    Parameters Windows 10 Windows 11
    Released On July 15, 2015 June 24, 2021
    Functionality Based on Andriod Functionality Based on Mac Functionality
    TPM Runs on devices with TPM 1.2 or above Runs on devices with TPM 2.0
    RAM & Storage Runs on devices with a min 2 GB RAM & 32 GB Storage Runs on devices with a min 4 GB RAM & 64 GB Storage
    Start Menu The start menu is located at the left of the screen. The start menu is located at the center of the screen and can be adjusted.
    Microsoft Support Microsoft will end the support for Windows 10 in 2025 Microsoft will continue the support for Windows 11 till the next version is launched.
